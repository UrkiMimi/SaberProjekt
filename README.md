# SaberProjekt
(VERY WIP)

This is a fork of the SaberFactory AssetProject due to it being unmaintained. Features and such are subjected to change! At the moment, do not use this as your main project to create Custom Sabers.

### Unity stuff
This project is built on Unity 2019.4.18 instead of Unity 2019.3.15. This might hurt model compatibility a little bit with really old versions but not too much. (Basically 1.13.2 and older)
